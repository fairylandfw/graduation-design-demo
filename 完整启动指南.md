# 网络舆情分析平台 - 完整启动指南

## 🎯 系统状态

✅ 后端代码已就绪
✅ 数据库已初始化（13条数据）
✅ 所有依赖已安装
⏳ 需要手动启动服务

## 🚀 启动步骤（推荐）

### 第1步：启动后端

双击运行：
```
backend\start-backend.bat
```

或手动运行：
```bash
cd backend
venv\Scripts\activate
python run.py
```

等待看到：
```
* Running on http://127.0.0.1:5000
```

### 第2步：启动前端

**打开新的命令行窗口**，双击运行：
```
frontend\start-frontend.bat
```

或手动运行：
```bash
cd frontend
npm run serve
```

首次运行会安装依赖（约3-5分钟），等待看到：
```
App running at:
- Local: http://localhost:8080
```

### 第3步：访问系统

浏览器打开：http://localhost:8080

## 📊 你将看到

1. **情感分布饼图** - 13条数据的情感分布
   - 正面、中性、负面比例

2. **情感趋势图** - 时间序列分析
   - 最近7天的情感变化

3. **热点词云** - 关键词可视化
   - 高频词汇展示

4. **文章列表** - 舆情数据
   - 标题、来源、情感标签
   - 来自微博、抖音、B站、小红书

## 🔄 更新数据

随时可以采集新数据：
```bash
cd backend
venv\Scripts\activate
python auto_crawler.py
```

每次运行会采集12条新数据，自动去重。

## 🛠️ 故障排除

### 问题1：后端启动失败

**检查MySQL服务**
```bash
net start | findstr MySQL
```

如果未启动：
```bash
net start MySQL80
```

**检查端口占用**
```bash
netstat -ano | findstr :5000
```

### 问题2：前端启动失败

**清理并重新安装**
```bash
cd frontend
rmdir /s /q node_modules
npm install
```

**检查端口占用**
```bash
netstat -ano | findstr :8080
```

### 问题3：前端显示空白

1. 确认后端已启动（访问 http://localhost:5000/api/data/articles）
2. 检查浏览器控制台（F12）是否有错误
3. 刷新页面（Ctrl+F5）

### 问题4：数据库连接失败

检查配置文件 `backend/config/config.py`：
```python
'mysql+pymysql://root:123456@localhost:3306/sentiment_db?charset=utf8mb4'
```

确认密码是否正确（当前：123456）
