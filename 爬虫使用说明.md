# 最终版爬虫使用说明

## ✅ 可用的爬虫

### 1. enhanced_crawler.py（推荐）

**数据来源**:
- ✅ 百度热搜（20条真实数据）
- ✅ 今日头条（10条模拟热点）
- ✅ 网易新闻（5条新闻话题）

**使用方法**:
```bash
cd backend
venv\Scripts\activate
python enhanced_crawler.py
```

**特点**:
- 稳定可靠，不依赖需要认证的API
- 包含真实百度热搜数据
- 模拟数据有明确的情感倾向（正面/负面/中性）
- 每次采集35条数据

## 🔧 故障排除

### 问题：情感分析还是全中性
**解决**：
1. 确认已重启后端服务
2. 运行 `enhanced_crawler.py` 采集新数据
3. 新数据包含明确情感倾向的内容

### 问题：百度热搜爬取失败
**解决**：
1. 检查网络连接
2. 检查是否被防火墙拦截
3. 使用模拟数据部分（今日头条、网易新闻）
