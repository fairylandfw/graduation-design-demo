# 使用说明

## 一键启动

1. 双击 `start.bat`
2. 等待自动安装依赖和采集数据
3. 浏览器会自动打开 http://localhost:8080

就这么简单！

## 启动过程说明

### 第1步：准备环境
- 创建Python虚拟环境
- 安装后端依赖（Flask、Jieba等）

### 第2步：初始化数据库
- 自动创建SQLite数据库
- 创建所需的数据表

### 第3步：采集数据
- 从微博、抖音、B站、小红书采集热点内容
- 自动进行情感分析
- 提取关键词和话题

### 第4步：启动后端
- 启动Flask服务器
- 监听端口5000
- 提供API接口

### 第5步：启动前端
- 安装前端依赖（首次运行）
- 启动Vue开发服务器
- 监听端口8080

## 查看数据

访问 http://localhost:8080 可以看到：

1. **情感分布饼图** - 显示正面/中性/负面舆情比例
2. **情感趋势图** - 显示最近7天的情感变化
3. **热点词云** - 显示高频关键词
4. **最新舆情列表** - 显示最新采集的文章

## 更新数据

### 自动更新
系统会定期自动采集新数据（可在配置中设置）

### 手动更新
打开命令行，运行：
```bash
cd backend
venv\Scripts\activate
python auto_crawler.py
```

## 停止服务

关闭"Backend Service"和"Frontend Service"两个命令行窗口即可。

## 自定义配置

### 修改采集平台
编辑 `backend/auto_crawler.py`，在 `get_sample_data()` 函数中添加或修改数据源。

### 修改情感词典
编辑 `data/dictionaries/` 目录下的文件：
- `positive.txt` - 添加正面词汇
- `negative.txt` - 添加负面词汇
- `degree.txt` - 添加程度副词及权重
- `negation.txt` - 添加否定词

### 修改端口
- 后端端口：编辑 `backend/run.py`
- 前端端口：编辑 `frontend/package.json` 中的 serve 脚本

## 常见问题

### Q: 启动时提示Python未找到
A: 请安装Python 3.8+，下载地址：https://www.python.org/downloads/

### Q: 启动时提示Node.js未找到
A: 请安装Node.js 14+，下载地址：https://nodejs.org/

### Q: 端口被占用
A: 检查5000和8080端口是否被其他程序占用，关闭占用程序或修改配置中的端口

### Q: 依赖安装失败
A: 脚本已配置国内镜像源，如仍失败，请检查网络连接

### Q: 浏览器显示空白
A: 等待前端服务完全启动（约30秒），然后刷新页面

### Q: 没有数据显示
A: start.bat会自动采集数据，如果没有，手动运行 `python backend/auto_crawler.py`
