# 常见问题解决方案

## 问题1：localhost拒绝连接

### 症状
- 前端显示"连接被拒绝"
- 无法访问 http://localhost:5000

### 原因分析
1. 后端服务未启动
2. 端口被占用
3. 数据库连接失败
4. 防火墙阻止

### 解决方案

#### 步骤1：检查后端是否启动
查看运行 `start-backend.bat` 的窗口，是否有错误信息

#### 步骤2：检查端口占用
```bash
netstat -ano | findstr :5000
```
如果端口被占用，可以：
- 关闭占用端口的程序
- 或修改后端端口（编辑 `backend/run.py`）

#### 步骤3：检查数据库连接
确保MySQL已启动：
```bash
# 检查MySQL服务
net start | findstr MySQL
```

如果MySQL未启动：
```bash
net start MySQL80  # 服务名可能不同
```

#### 步骤4：简化配置（临时方案）
如果暂时无法配置MySQL和Redis，可以修改为使用SQLite：

编辑 `backend/config/config.py`，将数据库配置改为：
```python
SQLALCHEMY_DATABASE_URI = 'sqlite:///sentiment.db'
```

然后重新启动后端。

## 问题2：网站没有数据

### 原因
数据库初始为空，需要添加数据

### 解决方案

#### 方案A：添加示例数据（推荐用于测试）

运行示例数据脚本：
```bash
add-sample-data.bat
```

这会添加10条示例新闻数据，包含：
- 正面新闻（如便民服务、科技创新）
- 负面新闻（如质量问题、交通拥堵）
- 中性新闻

#### 方案B：运行爬虫采集数据

**简单模式（演示）**：
```bash
run-crawler.bat
```
选择选项1，会添加演示数据

**自定义爬虫**：
1. 编辑 `backend/run_crawler.py`
2. 修改 `crawl_custom_website` 函数
3. 根据目标网站的HTML结构修改CSS选择器
4. 运行 `run-crawler.bat`，选择选项2

#### 方案C：手动添加数据

通过API接口添加：
```bash
curl -X POST http://localhost:5000/api/data/articles ^
  -H "Content-Type: application/json" ^
  -d "{\"title\":\"测试标题\",\"content\":\"测试内容\"}"
```

## 问题3：情感分析不准确

### 原因
词典不够完善

### 解决方案

扩展情感词典：

1. 编辑 `data/dictionaries/positive.txt` 添加正面词
2. 编辑 `data/dictionaries/negative.txt` 添加负面词
3. 编辑 `data/dictionaries/degree.txt` 添加程度副词
4. 重启后端服务

示例：
```
# positive.txt
优秀
卓越
完美
给力

# negative.txt
糟糕
失望
愤怒
差劲

# degree.txt
非常 2.0
极其 2.0
稍微 0.8
```

## 问题4：前端显示空白

### 检查步骤

1. **检查浏览器控制台**
   - 按F12打开开发者工具
   - 查看Console标签是否有错误

2. **检查网络请求**
   - 在Network标签查看API请求
   - 确认请求状态码是否为200

3. **检查后端API**
   直接访问：http://localhost:5000/api/data/articles
   应该返回JSON数据

## 问题5：依赖安装失败

### Python依赖安装慢
使用国内镜像：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### npm安装慢
使用淘宝镜像：
```bash
npm config set registry https://registry.npmmirror.com
npm install
```

### 特定包安装失败
逐个安装，找出问题包：
```bash
pip install Flask
pip install Flask-CORS
# ... 逐个安装
```

## 问题6：数据库初始化失败

### 手动初始化

1. 进入backend目录
```bash
cd backend
venv\Scripts\activate
```

2. 运行Python命令
```python
python
>>> from app import create_app, db
>>> app = create_app()
>>> with app.app_context():
...     db.create_all()
>>> exit()
```

## 问题7：Redis连接失败

### 临时方案
注释掉Redis相关代码（仅用于测试）：

编辑 `backend/app/__init__.py`，注释Redis初始化：
```python
# redis_client = redis.Redis(...)
```

### 永久方案
安装并启动Redis：
1. 下载：https://github.com/tporadowski/redis/releases
2. 解压后运行 `redis-server.exe`
