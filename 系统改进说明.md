# 系统改进说明

## ✅ 已完成的改进

### 1. 真实网络爬虫

**新增文件**: `backend/real_crawler.py`

**功能**:
- ✅ 爬取微博热搜（公开API）
- ✅ 爬取知乎热榜（公开API）
- ✅ 爬取百度热点（网页抓取）

**使用方法**:
```bash
cd backend
venv\Scripts\activate
python real_crawler.py
```

**特点**:
- 真实的网络数据，不是模拟数据
- 自动去重，避免重复采集
- 包含真实的浏览量、评论数等统计信息
- 每次可采集30-45条真实热点数据

### 2. 优化情感分析算法

**改进内容**:

1. **提高基础权重**
   - 正面词权重: 1.0 → 1.5
   - 负面词权重: -1.0 → -1.5

2. **扩大程度副词影响范围**
   - 检查范围: 前1个词 → 前2个词
   - 支持多个程度副词叠加

3. **增强否定词效果**
   - 检查范围: 前3个词 → 前4个词
   - 否定效果: ×(-1) → ×(-1.2)

4. **改进归一化方法**
   - 旧方法: 除以总词数
   - 新方法: 除以情感词数量 + sigmoid函数增强

5. **降低中性阈值**
   - 正面阈值: 0.1 → 0.05
   - 负面阈值: -0.1 → -0.05
   - 提高区分度，减少中性判断

6. **扩展情感词典**
   - 正面词: 30个 → 90+个
   - 负面词: 30个 → 90+个
   - 程度副词: 11个 → 25个

**效果对比**:

旧算法:
- "这个产品非常好" → neutral (0.0213)
- "服务太差了" → neutral (-0.0698)

新算法:
- "这个产品非常好" → positive (0.3+)
- "服务太差了" → negative (-0.3+)

## 🔧 故障排除

### 问题1: 爬虫无法访问网站
- 检查网络连接
- 检查防火墙设置
- 某些网站可能需要代理

### 问题2: 情感分析仍然全是中性
- 确认已重启后端服务
- 检查词典文件是否更新
- 运行真实爬虫采集新数据

### 问题3: 数据重复
- 系统会自动去重
- 如需清空数据库重新开始:
```sql
TRUNCATE TABLE articles;
```
