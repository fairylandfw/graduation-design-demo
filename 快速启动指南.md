# 快速启动指南

## 方式一：使用启动脚本（推荐）

### 1. 分别启动后端和前端

**启动后端**（在第一个命令行窗口）:
```bash
start-backend.bat
```

**启动前端**（在第二个命令行窗口）:
```bash
start-frontend.bat
```

### 2. 一键启动（可能需要手动）
```bash
start.bat
```

## 方式二：手动启动

### 启动后端

1. 打开命令行，进入项目目录
```bash
cd sentiment-analysis-platform\backend
```

2. 创建虚拟环境（首次运行）
```bash
python -m venv venv
```

3. 激活虚拟环境
```bash
venv\Scripts\activate
```

4. 安装依赖（首次运行）
```bash
pip install -r requirements.txt
```

5. 启动后端服务
```bash
python run.py
```

后端将在 http://localhost:5000 启动

### 启动前端

1. 打开新的命令行窗口，进入前端目录
```bash
cd sentiment-analysis-platform\frontend
```

2. 安装依赖（首次运行）
```bash
npm install
```

3. 启动前端服务
```bash
npm run serve
```

前端将在 http://localhost:8080 启动

## 环境准备

### 1. 安装Python
- 下载: https://www.python.org/downloads/
- 版本: 3.8 或更高
- 安装时勾选 "Add Python to PATH"

### 2. 安装Node.js
- 下载: https://nodejs.org/
- 版本: 14 或更高
- 自动包含npm

### 3. 安装MySQL
- 下载: https://dev.mysql.com/downloads/mysql/
- 版本: 5.7 或更高
- 创建数据库:
```sql
CREATE DATABASE sentiment_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 4. 安装Redis
- Windows: https://github.com/tporadowski/redis/releases
- 下载并解压，运行 redis-server.exe

## 配置数据库

1. 复制环境变量配置文件
```bash
cd backend
copy .env.example .env
```

2. 编辑 `.env` 文件，修改数据库连接信息
```
DATABASE_URL=mysql+pymysql://root:你的密码@localhost:3306/sentiment_db?charset=utf8mb4
```

3. 初始化数据库表
```bash
python -c "from app import create_app, db; app = create_app(); app.app_context().push(); db.create_all()"
```

## 常见问题

### 1. Python命令不识别
- 确保安装时勾选了 "Add Python to PATH"
- 或使用 `py` 命令代替 `python`

### 2. npm install 很慢
使用淘宝镜像:
```bash
npm config set registry https://registry.npmmirror.com
```

### 3. pip install 很慢
使用清华镜像:
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 4. 数据库连接失败
- 确保MySQL服务已启动
- 检查用户名、密码是否正确
- 确认数据库 sentiment_db 已创建

### 5. Redis连接失败
- 确保 redis-server.exe 正在运行
- 检查端口 6379 是否被占用
