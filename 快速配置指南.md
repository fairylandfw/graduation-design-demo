# 快速配置指南

## 前提条件

确保已安装：
- ✅ Python 3.8+
- ✅ Node.js 14+
- ✅ MySQL 5.7+

## MySQL配置

### 1. 确保MySQL服务已启动

**Windows:**
```bash
net start MySQL80
```

### 2. 配置数据库密码

默认配置：
- 用户名: `root`
- 密码: `root`
- 数据库: `sentiment_db`

**如果你的MySQL密码不是root，请修改：**

编辑 `backend/config/config.py`，找到这一行：
```python
'mysql+pymysql://root:root@localhost:3306/sentiment_db?charset=utf8mb4'
```

将第二个 `root` 改为你的密码，例如：
```python
'mysql+pymysql://root:你的密码@localhost:3306/sentiment_db?charset=utf8mb4'
```

同时修改 `backend/init_db.py`：
```python
DB_PASSWORD = '你的密码'
```

## 启动系统

双击运行：
```
start.bat
```

系统会自动：
1. 安装Python依赖
2. 创建数据库和表
3. 采集初始数据
4. 启动后端和前端服务
5. 打开浏览器

## 访问地址

- 前端: http://localhost:8080
- 后端: http://localhost:5000

## 常见问题

### Q: 提示数据库连接失败
A: 检查MySQL服务是否启动，用户名密码是否正确

### Q: 提示端口被占用
A: 检查5000和8080端口是否被其他程序占用

### Q: 前端npm依赖安装慢
A: 首次启动会安装前端依赖，需要等待几分钟

## 手动步骤（如果自动启动失败）

### 后端
```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python init_db.py
python auto_crawler.py
python run.py
```

### 前端（新开命令行）
```bash
cd frontend
npm install
npm run serve
```

## 更新数据

```bash
cd backend
venv\Scripts\activate
python auto_crawler.py
```
